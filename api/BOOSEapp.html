Namespace BOOSEapp

Classes

AppArray

Represents a fixed-length array value in the BOOSE application. Stores numeric elements as AppValue objects.

AppBool

Represents a boolean value in the BOOSE application.

AppCanvas

Represents a simple drawing canvas used by the BOOSE application. Provides drawing primitives such as lines, shapes, text, and bitmap access. Maintains an internal cursor position for drawing operations.

AppCircle

Represents a command that draws an unfilled circle on the canvas using the specified radius (int or real) and the current cursor position.

AppCircleCmd

Command that draws a circle on the canvas.

AppClearCmd

Command that clears the canvas.

AppCommandFactory

Factory class responsible for creating executable drawing commands based on parsed command strings.

AppDrawToCmd

Command that draws a line from the current position to a specified position.

AppExecutionContext

Provides the execution context for the BOOSE application. Manages canvas access, output display, variables, methods, and expression evaluation.

AppFor

Executes a FOR loop: for var = start to end step stepValue ... end for Supports positive/negative step and prevents infinite loops.

AppIf

Represents an IFâ€“ELSE control structure in the BOOSE application. Evaluates a boolean condition and executes the corresponding block.

AppInt

Represents an integer value in the BOOSE application.

AppMethod

Represents a user-defined method in the BOOSE script. Example: method int mulMethod int one, int two mulMethod = one * two end method

AppMoveTo

Represents a command that moves the drawing cursor to a specified (x, y) position on the AppCanvas without drawing.

AppMoveToCmd

Command that moves the drawing cursor to a specified position without drawing a line.

AppNoOp

Represents a no-operation command in the BOOSE application. Optionally outputs a message when executed.

AppParser

Parses and executes BOOSE script text. Supports commands, variable declarations, assignments, output writing, arrays (peek/poke), and control structures (for/while/if) with nesting.

AppPeek

Implements the PEEK operation for the BOOSE application. Supports reading and printing variable values, reading array elements, and optionally assigning an array element to a target variable.

AppPen

Represents a command that changes the drawing pen color on the AppCanvas using RGB values.

AppPenCmd

Command that sets the drawing pen colour.

AppPoke

Implements the POKE operation for the BOOSE application. Supports assigning values to variables and setting elements inside arrays.

AppReal

Represents a real (double-precision) numeric value in the BOOSE application.

AppRect

Represents a command that draws a rectangle on the AppCanvas using the specified width and height.

AppRectCmd

Command that draws a rectangle on the canvas.

AppResetCmd

Command that resets the canvas state.

AppValue

Represents the base class for all value types used in the BOOSE application.

AppVariableStore

Manages storage and access of variables used in the BOOSE application. Provides methods for declaring, setting, and retrieving variables.

AppWhile

Executes a WHILE loop using the current execution context. Holds the loop condition and the body lines to execute.

DebugLogger

Provides a simple singleton-based logger for writing debug messages.

Form1

Main Windows Forms interface for the BOOSE drawing application. Handles program execution, command parsing, canvas rendering, and file operations for both programs and images.

ScriptEngine

ScriptEngine supports BOTH method styles:

STYLE A (BOOSE example): method int mulMethod int one, int two mulMethod = one * two end method

STYLE B (single-line header): method int mulMethod int one, int two mulMethod = one * two end method

Also supports common drawing commands: moveto, drawto, circle, rect, write (circle/rect default to outline, not filled).

Interfaces

IAppCommand

Defines a command that can be executed within the BOOSE application.
